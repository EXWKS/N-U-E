Nuxt 3 初期設定手順

npx nuxi init my-nuxt-app

すると Which package manager would you like to use? と訊かれるので npm を選択。

そのあと Initialize git repository? と訊かれるので Yes を選択（GitHubを利用するので）。

そのあと Would you like to install any of the official modules ときかれるが、特に今回は不要なので何も選択しない。

プロジェクトができあがるので、

cd my-nuxt-app

npm install

npm run dev

これで http://localhost:3000/ にWebブラウザでアクセスする。
